HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web
e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores
ou usando imagens e tabelas.

Outro conceito essencial em código é função, que permitem que você armazene um pedaço de código que
realiza uma simples tarefa dentro de um bloco, e então chama aquele código sempre que você precisar usá-lo
com um curto comando — em vez de ter que escrever o mesmo código multiplas vezes.
